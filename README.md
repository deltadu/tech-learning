# Dev Learning

Personal learning notes and code examples for C++, Python, and PyTorch/Deep Learning.

## Structure

```
├── cpp/                    # C++ fundamentals to modern features
│   ├── 01_basics/          # Functions, I/O, compiler warnings
│   ├── 02_data_structures/ # Vectors, maps, strings, enums/structs
│   ├── 03_oop/             # Classes and OOP patterns
│   ├── 04_memory/          # Pointers, references, smart pointers
│   ├── 05_modern_cpp/      # Lambdas, STL algorithms
│   └── 06_practical/       # File I/O, practical examples
│
├── python/                 # Intermediate to advanced Python
│   ├── 01_modern_python/   # Type hints, decorators, dataclasses, context managers
│   ├── 02_functional/      # Generators, itertools, functools
│   ├── 03_advanced/        # Asyncio, metaclasses, descriptors
│   ├── 04_performance/     # Concurrency, profiling
│   └── 05_testing/         # Pytest patterns
│
└── pytorch/                # Deep learning with PyTorch
    ├── 01_basics/          # Tensors, autograd
    ├── 02_neural_networks/ # nn.Module, training loops
    └── 03_transformers/    # Transformer architecture explained + implementation
```
